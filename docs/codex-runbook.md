## Task loop

1. Read relevant Mem0 context.
2. Read the Linear ticket.
3. Inspect repository state.
4. Confirm exact scope.
5. Implement.
6. Add or update tests.
7. Run relevant checks.
8. Fix failures.
9. Commit to feature branch.
10. Open or update PR.
11. Update Linear.
12. Update Mem0.
13. Summarise outcome.

Implementation tasks must not start before PRD ingestion.
